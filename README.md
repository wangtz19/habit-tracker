# 原子习惯 · Atomic Habits

基于《原子习惯》方法论的极简习惯打卡 PWA。完全本地存储，无需账号，无需服务器。

> 每天进步 1%，一年后强大 37 倍。 —— James Clear

## ✨ 核心特色

### 打卡与追踪
- 📔 **一秒打卡**：大圆形按钮 + 震动反馈，零负担
- 🌱 **原子习惯设计**：提示（Cue）· 习惯叠加 · 2 分钟版本 · 奖励，四要素加持
- 📝 **心情与备注**：长按卡片记一笔，5 档心情 + 200 字备注
- 🔥 **Streak 连击**：当前连击 + 历史最长，醒目高亮
- 🎯 **挑战模式**：21 天养成 / 66 天自动化 / 100 天精通 / 自定义天数
- 🔔 **每日提醒**：可为每个习惯设定提醒时间（需开启浏览器通知）

### 可视化
- 🟩 **GitHub 式热力图**：年度打卡草地，一目了然
- 📅 **月度日历**：点击补卡 / 取消，长按记笔记
- 📊 **完成率分析**：近 7 天 / 30 天 / 全部
- ⭕ **完成圈动画**：首页大圆环展示今日进度

### 分享与洞察
- 📸 **报告图片导出**：一键生成高清 PNG，分享朋友圈
- 🔗 **只读分享链接**：lz-string 压缩进 URL，对方无需账号即可查看
- ✨ **关联性洞察**：基于 Pearson 相关系数自动发现
  - 习惯之间的协同模式（"完成 A 的日子也更常完成 B"）
  - 心情与习惯的关联（"运动的日子心情高 0.8 分"）
  - 最佳完成时段（"星期六最易完成 88%"）
- 📅 **周报 / 月报**：每日柱状图 + 习惯排行 + 完美日统计
- 🏆 **17 个成就徽章**：5 大类别，解锁时右上角弹出庆祝

### 体验
- 🎨 **22 个精选模板**：基于循证心理学，按 8 大类分组
- 🔍 **搜索与筛选**：按名字搜，按 8 分类筛
- ↕️ **拖拽排序**：桌面端拖动，手机端长按拖
- ⌨️ **键盘快捷键**：`1-9` 直接打卡，`N` 新建，`?` 帮助
- 🗄 **归档管理**：旧习惯归档不删，随时恢复
- 🌗 **深色模式**：跟随系统 / 手动切换

### 数据与隐私
- 💾 **完全本地**：IndexedDB 存储，不上传任何服务器
- 📦 **JSON 导入导出**：永远拥有自己的数据
- 🛡 **分享链接不经服务器**：数据压缩编码在 URL 里
- 📱 **PWA 可装机**：iOS / Android 桌面图标，离线可用

## 🚀 快速开始

```bash
npm install
npm run dev
```

打开 http://localhost:3000

### 推荐试玩流程

1. 进入「**模板**」页 → 一键添加 3-5 个习惯
2. 回到「**今日**」→ 点大圆形按钮打卡
3. **长按**习惯卡片 → 记录心情和备注
4. 点开任一习惯 → 看挑战进度、热力图、月历
5. 底部「**统计**」→ 体验三个亮点：
   - ✨ 关联洞察（需 14 天数据）
   - 🏆 成就徽章
   - 📈 回顾报告（试两个右上角分享按钮）
6. **桌面端按 `?`** 查看键盘快捷键

## 🛠 技术栈

- **框架**：Next.js 14（App Router）+ React 18 + TypeScript
- **样式**：Tailwind CSS（深色模式）
- **状态**：Zustand
- **存储**：IndexedDB（idb）
- **日期**：date-fns
- **图标**：lucide-react
- **截图**：modern-screenshot（iOS Safari 友好）
- **压缩**：lz-string（分享链接编码）

## 🎨 设计哲学

不追求功能丰富，追求：

1. **打卡极快**：1 秒完成，不打扰
2. **方法论加持**：基于《原子习惯》四步法（提示·渴望·反应·奖励）
3. **数据归己**：本地优先，导出自由，分享无需服务器
4. **视觉激励**：热力图、连击数、完成率、徽章全方位激励
5. **诚实洞察**：基于真实数据的关联分析，不打鸡血

## 🔒 隐私

- 所有习惯和打卡数据保存在浏览器 IndexedDB
- 不会上传到任何服务器
- 分享链接的数据完全编码在 URL 里，不经过中转
- 分享时可选择是否包含心情数据
- 备注、提醒等敏感字段永不出现在分享链接中
